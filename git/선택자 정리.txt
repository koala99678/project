1. 기본 선택자의 종류
-태그 :태그명
-클래스 : .클래스명
-아이디 : #아이디 명
-공통 : *

2.조합선택자
-자손 :조상 자손 -띄어쓰기로 구분
-자식 :부모>자식 ->로 구분
-본인 :선택자선택자 -선택자를 붙여 사용
-형제 :
	1. 형제+형제 - 서로 붙어있는 형제
	2. 형제~형제 - 서로 형제 관계면 됨
-그룹화 : , -여러 선택자에 같은 속성을 적용하기 위해 사용
-속성 선택자 : [] -태그의 속성을 이용
-가상 선택자 :  :  -특정 상황이나 동작이 있을 때 사용
	
3.선택자 우선순위
-한 태그에 여러 선택자가 있는데 한 속성이 여러 선택자에 있는 경우
	!inoirant > id > calss > tag
-우선순위가 같은 경우는 나중에 적용되는 선택자가 적용
 
4. 속성들
-속성 : 값 ; 형태로 작성해야함

5.css 적용 방법
-style 속성을 이용하는 방법(비추- 우선순위 높음)
-style 태그를 이용하는 방법 -head태그 안에
-css파일을 생성하여 연결하는 방법 -head태그 안에 link를 이용
