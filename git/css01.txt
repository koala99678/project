선택자 :css를 적용하기 위한 요소

-태그 : 태그명만 사용

p{   css처리;}

-아이디:아이디명 앞에 #을 붙인다.

아이디가 id인 경우
#id{            css처리; }

-클래스: 클래스명 앞에 .을 붙인다,

클래스명이 nav인 경우
.nav{             css처리; }







css 적용 방법

1. 태그의 styel속성을 이용하여 직접 적용하는 방법. 우선순위가 가장높다.
왜냐하면 해당 테그 하나만 적용됨으로 (비추)

2.style 태그를 이용하는 방법, head 태그 안에 작성

3.css 파일을 만들어서 사용
