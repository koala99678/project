backfround : 이미지 반복 위치 ;

backfround-image :배경이미지 가져오기

backfround-repeat:배경이미지 반복여부

backfround-position :배경이미지 시작위치 설정

backfround-size :돋보기 기능

backfround-attachment : 이미지를 고정으로 할지 스크롤에 이동시킬지 결정

backfround-clip : padding을 제외하고 배경이미지를 넣을지 결정, padding 을 제외하고 넣는 경우 
padding부분을 비워둔다

backfround-origin :padding을 제외하고 배경이미지를 넣을지 결정, padding 을 제외하고 시작하는는 경우 
padding부분을 이미지로 채운다


border-style : 테두리 종류
border-width : 테두리 굵기
border-color : 테두리 색상
border : 종류 굵기 색상,입력하는 순서는 바뀌어도 상관 없음 (서로 단위가 다르기 때문에)


태그 = margin + border + padding + content

기본 width와 height는 content의 가로와 세로 상황에 따라서 width, height의 범위가 달라 질 수 있다.
box-sizing : border-box를 설정ㅎ면
width는 border - padding + content

margin : 박스의 시작 위치와 주변박스의 위치를 정하는 여백
